# Setup

Walk the user through any required first-time configuration.

## Steps

### 1. Check prerequisites

```bash
# Verify required tool is installed
your-tool --version
```

If not installed, guide the user to install it.

### 2. Prompt for configuration

Ask the user for any required values (credentials, endpoints, etc.).

### 3. Write configuration

Write config to the appropriate location.

### 4. Test the connection

```bash
your-tool --test
```

### 5. Confirm success

Print a clear success or failure message.
